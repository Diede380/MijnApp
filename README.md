# HuisVoorraad

## Doel

HuisVoorraad is een app om de voorraad in huis bij te houden.

### Functies (versie 1)

- Product toevoegen
- Houdbaarheidsdatum instellen
- Waarschuwing voordat een product verloopt
- Overzicht van producten per categorie
- Home-scherm met producten die bijna verlopen

### Toekomstige functies

- Barcode scannen
- Synchronisatie tussen meerdere telefoons
- Boodschappenlijst
- Donkere modus
